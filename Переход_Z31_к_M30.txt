 ПЕРЕХОД ОТ Z31 К M30 (по прошивкам ЭБУ)
 Z31: 88_Z31turbo32k.bin (300ZX VG30ET 88-89)
 M30: infiniti90m30at (1).bin (Infiniti M30 90-92, VG30E АКПП)
 Главный итог: M30 = Z31 + поддержка ДАТЧИКА ДЕТОНАЦИИ + реорганизация MAF-зоны.

==============================================================================
 1. ЧТО ОСТАЛОСЬ ОДИНАКОВЫМ (разметка совпадает по адресам)
==============================================================================
 Блок 0x7A70-0x7FFF — практически 1:1. Одни и те же адреса и смысл:

   0x7C00  Primary Timing Map            (карта зажигания, градусы)
   0x7D00  Primary Fuel Map              (топливо, AFR, флаг 0xC0 = замкн. цикл)
   0x7B30  Water Temp Sensor Conversion  (термистор, X-50 °C)
   0x7B50  Ignition Dwell (Battery)
   0x7B60  Ignition Dwell (RPM)
   0x7AF0  TP Scale - Fuel
   0x7B00  RPM Scale - Fuel
   0x7B10  TP Scale - Timing
   0x7B20  RPM Scale - Timing
   0x7E20  Cold Start Enrichment
   0x7E80  TTP Min / 0x7E90 TTP Max
   0x7EB0  Warm Start Idle Increase
   0x7EC0  Cold Start Timing Advance
   0x7F30  Cold Start Idle Increase
   0x7F2B  Injector K Value
   0x7F88  Injector Latency
   0x7F91  O2 Sensor Feedback Control
   0x7FA5  Top Speed Limit
   0x7FB4  Rev Limiter

 Кодировки тоже общие: AFR-формула топлива (1881.6/(X±..)), флаг 0xC0,
 16-бит экспонента VQ, спад термистора ТОЖ. => XDF переносится между ними.


==============================================================================
 2. ЧТО НОВОЕ В M30 (не было у Z31)
==============================================================================
 2.1 ДЕТОНАЦИОННАЯ ПАРА (из-за нового датчика детонации):
       0x7800  Knock Timing Map   — угол УБРАН до -12° в средней зоне
       0x7400  Knock Fuel Map     — смесь БОГАЧЕ (13.6 AFR vs 14.7)
     При детонации ЭБУ переключается: поздний угол + обогащение.
     У Z31 датчика детонации нет -> только Primary, без Knock-карт.

 2.2 СТРУКТУРНЫЙ БЛОК 0x6A00-0x6EFF (~600 б, НЕ в XDF, неразобран):
       4 почти одинаковых записи по ~134 б (0x6B00/0x6C00/0x6D00/0x6E00)
       с инкрементным индексом. У Z31 по этому адресу — КОД (данных нет).
       Гипотеза: данные под новое железо/диагностику. Нужен дизассемблер.

 2.3 Карта памяти: код у M30 лежит с 0x0000 (у Z31 первые 16КБ пусты,
     код с 0x4000).


==============================================================================
 3. ЧТО УШЛО (было у Z31, в M30 нет)
==============================================================================
   0x79B0  Вторая VQ/MAF-кривая (16-бит)  -> в M30 заливка 0xFF.
   0x7AB0  4 отдельные оси                -> в M30 это место заняла основная
                                             VQ-таблица (переехала, см. п.4).
   0x7200  Мелкие скаляры                 -> в M30 заливка 0x3F.
 Все три — из MAF/VQ-окрестности; M30 эту область упростил до одной VQ.


==============================================================================
 4. ЧТО СМЕСТИЛОСЬ / ИЗМЕНИЛОСЬ
==============================================================================
   VQ Table:   Z31 @0x7A30  ->  M30 @0x7A70 (сдвиг ~+0x40), и осталась ОДНА
               (у Z31 было две VQ-кривые, вторая на 0x79B0 — ушла).

   Числа-скаляры:
       параметр            Z31        M30
       Rev Limiter         6450       6250 об/мин
       Injector K          607        345        (другие форсунки)
       Top Speed Limit     ~220 км/ч  отключён
       Cold Start Tim.Adv  +25°       +30°
       O2 Feedback порог   10 °C      10 °C       (без изменений)


==============================================================================
 5. НЕРАСПОЗНАННОЕ — ОБЩЕЕ ДЛЯ ОБЕИХ
==============================================================================
   0x7ED9-0x7F2A (~82 б)  "шумовой" блок — тёмный и у Z31, и у M30
                          (тот же адрес/структура, значения разные).
   0x7E00-0x7EAF          кластер коррекционных кривых — без имён в обоих XDF.
   0x7BA0 / 0x7BC0-0x7BFF индексные ряды и кривые — без имён в обоих.
   Если расколоть их на ОДНОЙ прошивке — закроется сразу для обеих.


==============================================================================
 ВЫВОД
==============================================================================
 Z31 -> M30 это НЕ переписывание, а доработка той же платформы ECCS:
   + добавили датчик детонации -> добавили Knock-пару карт (зажигание + топливо);
   + добавили структурный блок 0x6A00 (под новое железо?);
   - убрали вторую VQ-кривую и отдельные оси, упростив MAF-зону до одной VQ;
   ~ подвинули VQ и перенастроили скаляры (отсечка, форсунки, лимит скорости).
 Базовая разметка калибровки (карты, тарировки, оси, скаляры) — общая.
 Метод распознавания (сигнатуры 0xC0/16-бит VQ/термистор/AFR) сработал на M30
 вслепую; XDF лишь подтвердил и дал имена (включая Knock-карты).
