==============================================================================
 РАСШИФРОВКА R32_GTS4_stock_ROM_MEC-R214B.bin ПО RB20det.xdf
 Nissan Skyline R32 GTS4, RB20, ЭБУ MEC-R214B.  XDF: legacy 1.10
 ВАЖНО: bin от GTS4 (атмо), XDF от RB20DET (турбо) — часть турбо-карт ПУСТЫЕ.
 BinSize=0x8000, BaseOffset=0 -> адрес XDF = смещение в файле.

==============================================================================
 ЧАСТЬ 1. ВСЕ ЭЛЕМЕНТЫ XDF (с формулами и декодированными значениями)
==============================================================================

--- КАРТЫ 16x16 (оси: RPM x TP) ----------------------------------------------
 0x7400  Primary Timing Map      eq 1.0*X        -> 4..154 (градусы + флаг-столбцы)
 0x7C00  Knock Timing Map        eq 1.0*X        -> идентична Primary (в этой прошивке)
 0x7D00  Primary Fuel Map        AFR: open 1881.6/(X+128), closed 1881.6/(X-64)
 0x7500  Secondary Fuel Map      AFR: 14.7/(1+X/128)   -> 5.8..14.3
   (топливо = AFR, флаг 0xC0 = замкнутый цикл — как у наших VG30.)

--- ПУСТЫЕ КАРТЫ (0xFF) — у GTS4 не используются ----------------------------
 0x7000  Knock Fuel Map (Open/Closed)          ПУСТО
 0x7100  5th gear Fuel Map (Open/Closed)       ПУСТО
 0x7200  5th Gear Knock Fuel Map (Open/Closed) ПУСТО
 0x6AD1  ROM ID                                ПУСТО (0xFF)

--- ТАРИРОВКИ / ОСИ ----------------------------------------------------------
 0x7A70  VQ Table (ДМРВ)         64x1 16-бит, экспонента 125..65535
 0x7B30  Water Temp Conversion   32x1, X-50 -> -50..205 °C
 0x7AF0  TP Scale - Fuel    X*0.125 -> 1.5..13.0
 0x7B00  RPM Scale - Fuel   50*X    -> 400..6400 RPM
 0x7B10  TP Scale - Timing  X*0.125 -> 1.5..13.0
 0x7B20  RPM Scale - Timing 50*X    -> 400..6400 RPM
 0x7B50  Ignition Dwell (Battery)  12.5..99.6 %
 0x7B60  Ignition Dwell (RPM)      5.0..76.9 %

--- ХОЛОСТОЙ ХОД / ПУСК / КОРРЕКЦИИ -----------------------------------------
 0x76B0  Base Idle RPM           32x1, X*12.5 -> 425..3187 RPM  (ЦЕЛЕВЫЕ ОБОРОТЫ ХХ vs темп)
 0x7E20  Cold Start Enrichment   16x1 -> 4..78 ms
 0x7E80  TTP Min                 16x1 -> 8 ms
 0x7E90  TTP Max                 16x1 -> 104..184 ms
 0x7EB0  Warm Start Idle Increase 16x1 -> 0..1900 RPM
 0x7F30  Cold Start Idle Increase 16x1 -> 0..1900 RPM
 0x7EC0  Cold Start Timing Advance 16x1 -> 0..20 град BTDC
 0x7B90  Timing retard versus temperature  16x1 -> все 0 (выключено в этой прошивке)
 0x7960  Air Flow Limiter        16x1 -> 128 (плоская)

--- ДЕТОНАЦИЯ / ОКТАН --------------------------------------------------------
 0x79E3  High Octane (Regular) Knock Coeff.  3x3 -> 0..5
 0x7FE0  Low Octane Knock Coeff.            3x3 -> 0..15

--- VTC (управление фазами распредвала) -------------------------------------
 0x79F5  VTC Solenoid Switch ON rpm   = 450 RPM
 0x79F6  VTC Solenoid Switch OFF rpm  = 6250 RPM
 0x79F3  VTC Minimum coolant temp     = 75 °F
 0x79F2  VTC Minimum actuation speed / 0x79F7 Minimum TP

--- ВЕНТИЛЯТОРЫ ОХЛАЖДЕНИЯ (пороги) ------------------------------------------
 0x7442/0x7443  Fan temp thresholds (°F) ; 0x7444/0x7445 задержки
 0x7F75/0x7F76  Fan speed thresholds (MPH)   [у GTS4 = 0xFF, не заданы]

--- СКАЛЯРЫ ------------------------------------------------------------------
 0x7FB4  Rev Limiter            = 7800 RPM
 0x7FA4  Rev Limit 2            = 7200 RPM
 0x7FE9  "parked rev limiter"   = 6050 RPM (на стоянке/нейтрали?)
 0x7FA5  Top Speed Limit        = 111.85 MPH (~180 км/ч)
 0x7F2B  Injector K Value       = 257
 0x7F88  Injector Latency       = 0.76 ms
 0x7F91  O2 Feedback Control    = 30 °C (порог входа в замкнутый цикл)
 0x7F92  O2 RICH Voltage Limit  = 0.61 В
 0x7F93  O2 LEAN Voltage Limit  = 0.23 В
 0x7FC8  Map Switching Constant = 4
 0x7EE0  FLAG: O2 sensor disable
 0x7761  FLAG: Fuel Map Correction Disable


==============================================================================
 ЧАСТЬ 2. КОНКРЕТНЫЕ ЧИСЛА ЭТОЙ ПРОШИВКИ (R32 GTS4)
==============================================================================
   Отсечка (Rev Limiter):      7800 об/мин ; Rev Limit 2: 7200 ; «стоянка»: 6050
   Лимит скорости:             ~180 км/ч (111.85 MPH)
   Целевые ХХ (Base Idle):     425..3187 об/мин по температуре (32 точки)
   Injector K:                 257 ; задержка форсунки: 0.76 мс
   Вход в замкнутый цикл O2:    от 30 °C ; пороги O2: rich 0.61В / lean 0.23В
   VTC включается:             450..6250 об/мин, от 75 °F ОЖ


==============================================================================
 ЧАСТЬ 3. ВАЛИДАЦИЯ / ЗАМЕЧАНИЯ
==============================================================================
 - Тарировка ОЖ (-50..205 °C), VQ-экспонента, оси (400-6400 об/мин) — осмысленны.
 - Knock Timing = Primary Timing (идентичны): в этой прошивке детонационная карта
   зажигания совпадает с основной (на NA-калибре отдельного отката нет/не задан).
 - Карты 0x7000/0x7100/0x7200 (Knock Fuel, 5th gear) — ПУСТЫЕ (0xFF): bin GTS4
   их не использует. Это и есть несовместимость bin(GTS4) vs XDF(RB20DET).
 - В этой прошивке также есть: VTC (управление фазами), 5th-gear карты,
   knock-коэффициенты, Map Switching Constant, Rev Limit 2.
 - Часть скаляров (вентиляторы по скорости, VTC max temp) = 0xFF (не заданы у GTS4).
