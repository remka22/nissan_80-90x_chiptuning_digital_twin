==============================================================================
   ГЛАВНОЕ — ВОТ ЭТОТ БАЙТ
==============================================================================

   ┌───────────────────────────────────────────────────────────┐
   │                                                           │
   │   FILE OFFSET (в .bin):     0x37D0                         │
   │   CPU-АДРЕС:                $B7D0                          │
   │   ТИП:                      1 байт  (uint8)               │
   │   ЧТО ЭТО:                  операнд инструкции ADDA #$XX   │
   │                             (опкод 0x8B стоит на 0x37CF —  │
   │                              ЕГО НЕ ТРОГАТЬ!)              │
   │                                                           │
   │   СТОК:                     0x40  (64)   ->  ~922 об/мин   │
   │   ПОНИЖЕНО (пример):        0x28  (40)   ->  ~750 об/мин   │
   │                                                           │
   └───────────────────────────────────────────────────────────┘

   Меньше байт  ->  ниже холостой.   Больше байт  ->  выше холостой.


==============================================================================
   ФОРМУЛА (как байт превращается в обороты)
==============================================================================

   В прошивке на $B7D0 цель ХХ собирается так:

        база $1422 = ( лукап(64 на ХХ)  +  БАЙТ )  x 32
        обороты    = база $1422  x  0.225

   Подставив:

        ОБОРОТЫ_ТЁПЛЫЕ  =  ( 64 + БАЙТ )  x  7.2

        БАЙТ  =  ОБОРОТЫ / 7.2  -  64

   Шаг одного значения байта  ≈  7.2 об/мин.


==============================================================================
   ТАБЛИЦА: БАЙТ -> ОБОРОТЫ
==============================================================================

      БАЙТ (hex)   БАЙТ (dec)   ТЁПЛЫЙ ХОЛОСТОЙ
      ----------   ----------   ---------------
        0x18           24           ~634
        0x20           32           ~691
        0x28           40           ~749   <- текущий патч
        0x30           48           ~806
        0x38           56           ~864
        0x40           64           ~922   <- СТОК (заводское)
        0x48           72           ~979
        0x50           80          ~1037

   (прогрев на холод прибавляется СВЕРХУ — эту базу не трогает)


==============================================================================
   ДЛЯ TUNERPRO  /  XDF  (определение параметра)
==============================================================================

   Title (имя):            Холостой ход — база (тёплый)
   Description:            Тёплая база целевых оборотов ХХ ($1422)
   Address (offset):       0x37D0
   Size:                   8 bit  (1 байт)
   Data Type:              uint8 (unsigned, без знака)
   Endianness:             — (1 байт)

   Equation (raw -> RPM):  (64 + X) * 7.2
       (упрощённо то же:   X * 7.2 + 460.8 )

   Inverse (RPM -> raw):   X = RPM / 7.2 - 64

   Units:                  RPM
   Min / Max (безопасно):  0x18 .. 0x58   (~634 .. ~1100 об)
   Decimal places:         0


==============================================================================
   ВАЖНО / ПРЕДОСТЕРЕЖЕНИЯ
==============================================================================

   * Это РЕАЛЬНАЯ перешивка ЭБУ: блок САМ начнёт хотеть новый холостой,
     контур РХХ подведёт обороты к нему. Не костыль.

   * ТОЛЬКО ОДИН БАЙТ — 0x37D0. Соседние не трогать:
        0x37CF = 0x8B (опкод ADDA, оставить)
        0x37D6.. = STD $1422 (оставить)

   * Топливо / смесь НЕ затрагивается: AFR ($141A) пишется ДО этого байта
     (на $B7CC), так что карта топлива остаётся как есть.

   * Прогревочная добавка (холодный ХХ) идёт отдельной таблицей 0x75A0 и
     складывается поверх базы — её этот байт не меняет.

   * Слишком низко (база < ~0x20) — мотор на тёплую может не удержать ХХ,
     если РХХ не хватит воздуха срезать/добавить. Проверять на ходу.

   * Числа оборотов — по нашему откалиброванному масштабу
     ($1422=4096 <-> ~922, сверено с реальной машиной ~930).
     Реальные обороты могут отличаться на ±пару десятков.
