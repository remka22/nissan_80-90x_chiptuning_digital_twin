==============================================================================
 ЧАСТЬ 1. ГЛАВНОЕ: XDF ПОДТВЕРДИЛ ГИПОТЕЗУ ПРО ДЕТОНАЦИЮ
==============================================================================
 У M30 появился датчик детонации -> появился АЛЬТЕРНАТИВНЫЙ "детонационный"
 калибр, ПАРОЙ (зажигание + топливо). Имена таблиц в XDF — дословно:

   0x7800  Knock Timing Map            <- детонационная зажигания (угол убран)
   0x7C00  Primary Timing Map          <- нормальная зажигания (угол больше)
   0x7400  Knock Fuel Map (Open/Closed)<- детонационная топливная (богаче)
   0x7D00  Primary Fuel Map (Open/Closed)<- нормальная топливная (стехиометрия)

 Наши числа сходились заранее:
   - Зажигание: 0x7C00 имел до +12 град БОЛЬШЕ угла, чем 0x7800 (в средней,
     детонационной зоне). => 0x7800 = откат угла при детонации. ВЕРНО.
   - Топливо: в средней зоне 0x7400 = богаче (открытый цикл), 0x7D00 = беднее
     (стехиометрия). => при детонации обогащение (охлаждение). ВЕРНО.

 Итог: при детонации ЭБУ делает ДВА действия — убирает угол И обогащает.
 Поэтому у M30 ДВЕ карты зажигания и ДВЕ топливных, а у Z31 (без датчика
 детонации) — по одной.


==============================================================================
 ЧАСТЬ 2. ВСЕ ТАБЛИЦЫ XDF (с формулами и диапазонами)
==============================================================================

--- КАРТЫ 16x16 (оси: RPM x TP) ----------------------------------------------
 0x7C00  Primary Timing Map      eq 1.0*X        -> 0..50 град (251 = флаг-байт)
 0x7800  Knock Timing Map        eq 1.0*X        -> 0..50 град (откат vs Primary)
 0x7D00  Primary Fuel Map        AFR: open 1881.6/(X+128), closed 1881.6/(X-64)
 0x7400  Knock Fuel Map          AFR: те же формулы (богаче в детонац. зоне)
   Топливо — это AFR (как на Z31), 0xC0-флаг ячейки = замкнутый цикл.

--- ОСИ (по 16 точек) --------------------------------------------------------
 0x7AF0  TP Scale - Fuel    X*0.125  -> 1.88..12.75
 0x7B00  RPM Scale - Fuel   50*X     -> 400..6000 RPM
 0x7B10  TP Scale - Timing  X*0.125  -> 1.88..13.75
 0x7B20  RPM Scale - Timing 50*X     -> 400..5600 RPM

--- ТАРИРОВКИ ДАТЧИКОВ -------------------------------------------------------
 0x7A70  VQ Table (ДМРВ/MAF)  64x1, 16-бит, экспонента 125..65535
 0x7B30  Water Temp Sensor Conversion  32x1, X-50 -> -50..205 °C (термистор)

--- DWELL --------------------------------------------------------------------
 0x7B50  Ignition Dwell Duty (Battery Voltage)  X*100/256 -> 12.5..99.6 %
 0x7B60  Ignition Dwell Duty (RPM)              0.33*X    -> 4.95..76.6 %

--- ХОЛОДНЫЙ ПУСК / КОРРЕКЦИИ ------------------------------------------------
 0x7E20  Cold Start Enrichment       X     -> 6..96 ms
 0x7E80  TTP Min                     X     -> 7..10 ms
 0x7E90  TTP Max                     X     -> 106..135 ms
 0x7EB0  Warm Start Idle Increase    X*50  -> 0..2150 RPM
 0x7EC0  Cold Start Timing Advance   X     -> 0..30 град BTDC
 0x7F30  Cold Start Idle Increase    X*50  -> 0..2150 RPM
 0x7960  Air Flow Limiter            X     -> все 255 (отключён)

--- СКАЛЯРЫ ------------------------------------------------------------------
 0x7F2B  Injector K Value            = 345  (16-бит)
 0x7F88  Injector Latency            = 0.75 ms
 0x7F91  O2 Sensor Feedback Control  = 10 °C (порог замкнутого цикла)
 0x7FA5  Top Speed Limit             = raw 255 -> отключён (без ограничения)
 0x7FB4  Rev Limiter                 = 6250 RPM


==============================================================================
 ЧАСТЬ 3. КОНКРЕТНЫЕ ЧИСЛА M30 (и отличия от Z31)
==============================================================================
   Отсечка:           6250 об/мин   (у Z31 было 6450)
   Лимит скорости:    отключён       (у Z31 ~220 км/ч)
   Injector K:        345            (у Z31 607 — другие форсунки)
   Доп. угол холодн.: до +30 град    (у Z31 +25)
   Порог O2:          10 °C          (как Z31)
   Задержка форсунки: 0.75 ms        (как Z31)


==============================================================================
 ЧАСТЬ 4. СОВПАДЕНИЕ РАЗМЕТКИ С Z31  (то, ради чего всё затевалось)
==============================================================================
 Блок 0x7A70-0x7FFF у M30 — ПОЧТИ 1:1 с Z31 по адресам:

   адрес     M30                          Z31                        совпало?
   0x7C00    Primary Timing Map           Primary Timing Map          ДА
   0x7D00    Primary Fuel Map             Primary Fuel Map            ДА
   0x7B30    Water Temp Sensor Conv.      Water Temp Sensor Conv.     ДА
   0x7B50    Dwell (Battery)              Dwell (Battery)             ДА
   0x7B60    Dwell (RPM)                  Dwell (RPM)                 ДА
   0x7AF0..  TP/RPM Scale x4              TP/RPM Scale x4             ДА
   0x7E20    Cold Start Enrichment        Cold Start Enrichment       ДА
   0x7E80..  TTP Min/Max, Idle Inc.       те же                       ДА
   0x7F2B..  Injector K/Lat, O2, Top,Rev  те же                       ДА (адреса)
   0x7A70    VQ Table                     VQ Table @0x7A30            СДВИГ (~+0x40)

 ОТЛИЧИЯ M30 от Z31:
   + ДЕТОНАЦИОННАЯ ПАРА: Knock Timing @0x7800 + Knock Fuel @0x7400 (нет у Z31).
   + Структурный блок @0x6A00 (повторяющиеся записи; нет у Z31; не в XDF).
   + Код лежит с 0x0000 (у Z31 первые 16КБ пустые, код с 0x4000).
   ~ VQ-таблица сдвинута (0x7A70 vs 0x7A30).

 ВЫВОД: со стороны калибровки M30 и Z31 — практически одна разметка
 (адреса Primary-карт, тарировок, осей, скаляров совпадают). M30 = Z31
 ПЛЮС детонационный калибр (из-за датчика детонации) и доп. структурный блок.
 XDF от M30 = XDF от Z31 + Knock-карты. Перенос определений между ними реален.
